# Drawing a Heart with Python Turtle

This is a small, artistic project that uses Python’s built-in **Turtle Graphics** module to draw a heart shape.  
It’s a simple yet fun way to explore **creative coding** and visualize art through basic programming.

---

## 🐢 About Turtle

The `turtle` module lets you control a “turtle” that moves around the screen, drawing lines as it goes — perfect for visual learners and creative minds.

---

## ✨ What This Project Does

- Draws a heart shape using Python Turtle 🧡  
- Demonstrates basic movement commands (`forward`, `left`, `circle`)  
- Combines math, geometry, and art in a simple way  
- A great beginner-friendly introduction to Python graphics
